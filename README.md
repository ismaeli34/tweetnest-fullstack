# tweetnest-fullstack

TweetNest is a full-stack web application that allows users to 

📰 View Tweets: Display real-time tweets from a user’s timeline.

✍️ Post Tweets: Compose and publish new tweets directly from the app.

💬 Reply & Retweet: Interact with existing tweets just like on Twitter.

❤️ Like Tweets: Like or unlike tweets with instant UI updates.

👤 Follow/Unfollow: Manage user connections from within the app.
